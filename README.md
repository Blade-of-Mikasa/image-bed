# image-bed

目录

```bash
images/
├── content/        # 博客 / 笔记 / 技术文档引用
│   ├── blog/
│   ├── notes/
│   └── projects/
├── bg/             # 背景图 / 壁纸
├── assets/         # 通用素材（icon / logo / 装饰图）
└── misc/           # 临时 / 未分类（定期清理）
```

格式 

`用途-内容(-序号).扩展名`

| ❌ 原文件名          | ✅ 新文件名                  |
| --------------- | ----------------------- |
| Edu 124 (2).png | notes-edu124-fig2.png   |
| bg_jpg (5).jpg  | bg-elden-ring-05.jpg    |
| FwTree.png      | algo-fw-tree.png        |
| TimeStamp.jpg   | blog-timestamp-demo.jpg |
